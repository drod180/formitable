# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

### Users

- `GET /users/new`
- `POST /users`
- `PATCH /users`

### Session

- `GET /session/new`
- `POST /session`
- `DELETE /session`


## JSON API

### Forms

- `GET /api/forms`
- `POST /api/forms`
- `GET /api/forms/:id`
- `PATCH /api/forms/:id`
- `DELETE /api/forms/:id`
- `GET /api/forms/:id/fields`
- `POST /api/forms/:id/fields`

### Fields

- `PATCH /api/fields/:id`
- `DELETE /api/fields/:id`
